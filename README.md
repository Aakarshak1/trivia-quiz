# Trivia Quiz

Trivia Quiz app built using `React`, `Redux`, `Material-UI` and `Styled Component`

- Clone the Repo
- Install all dependency using `npm install`

### Run `npm start`

Runs the app.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
